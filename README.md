# Automated Quiz Engine with PDF Certification Generation

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/bhargav9347/quiz-cert-engine)

## Start Instructions
1. Start XAMPP (Apache + MySQL)
2. Run `schema.sql` in phpMyAdmin or mysql cli
3. Configure `.env` if needed
4. Run `npm install`
5. Run `npm run dev`
6. Visit `http://localhost:3000`

Login with:
- admin@quiz.com
- Admin@123

---

## 🚀 Deployment (One-Click)

### Option 1: Render.com
1. Create a MySQL database on Render or a provider like **Aiven.io**.
2. Connect this GitHub Repo to **Render**.
3. It will automatically detect the `render.yaml` file.
4. Set your `DB_HOST`, `DB_USER`, and `DB_PASSWORD` in the Render Dashboard environment settings.

### Option 2: Railway.app
1. Click **New Project** -> **MySQL**.
2. Click **New** -> **GitHub Repo** -> `quiz-cert-engine`.
3. Railway will automatically link them!
